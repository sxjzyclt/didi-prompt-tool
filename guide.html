Overview

这个网站是干什么的

Didi Prompt Tool 是一个广告创意 Prompt 工作台。Prompt（提示词）就是你发给 AI 绘图或 AI 视频工具的文字说明。这个网站可以帮你把广告画面里的主体、场景、构图、风格、文案和负面规则整理成一段完整提示词。

小程序 Banner 活动页主视觉 电商促销海报 本地生活广告 游戏宣传图 旅游宣传图 短视频开屏图 文生视频首帧

Quick Start

第一次使用

  1. 打开在线网页：https://sxjzyclt.github.io/didi-prompt-tool/
  2. 点击页面上的“进入工作台”。
  3. 左侧选择一个预设，比如“小程序广告”或“电商促销”。
  4. 中间编辑每个模块里的内容。
  5. 右下方查看“完整提示词”。
  6. 点击“复制完整 Prompt”，粘贴到你常用的 AI 绘图或 AI 视频工具里。

Presets

选择预设

左侧是“预设中心”。你可以直接点击预设卡片，也可以用搜索框搜索关键词，比如：旅游、电商、游戏、科技、本地生活。

小建议

新手不要从空白开始写。先选一个最接近目标的预设，再改文案和画面内容，会快很多。

Modes

文生图和文生视频怎么选

文生图（Text-to-Image）

适合生成海报、Banner、主视觉、封面图。如果你只是想做一张广告图，优先选它。

文生视频（Text-to-Video）

适合生成短视频、开屏动态素材、镜头运动提示词。选择后会多出视频长度、镜头运动、动态强度等设置。

Fields

怎么填写内容

基础设定
给这条提示词起名字，并选择画面比例，比如 16:9、9:16、1:1、4:5。
画面内容
填写主体、场景、动作、服装、道具、构图和镜头。它决定 AI 画面里“有什么”。
视觉风格
填写商业插画、3D 卡通、轻写实广告、电影海报、光线、色彩和画质要求。
广告文案
填写主标题、副标题、优惠信息、按钮文案和完整标语。
负面规则
告诉 AI 不要生成乱码、错别字、水印、二维码、真实品牌 Logo 和个人信息。

Output

输出格式和复制按钮

输出格式

结构化中文最适合新手，分段清楚。一行中文适合单行输入框。SD 分段适合 Stable Diffusion 这类区分 Positive 和 Negative 的工具。

复制按钮

复制完整 Prompt最常用。复制 Positive只复制正向提示词。复制 Negative只复制负面提示词。

Backup

保存自己的预设和导出 JSON

如果你改出了一套常用模板，可以点击“保存为我的预设”。保存后，它会出现在左侧“我的预设”分类里。

重要提醒

“我的预设”保存在浏览器本地，不是在服务器上。换电脑、清理浏览器数据后可能会消失，建议定期点击“导出 JSON”备份。

  1. 导出 JSON：点击“导出 JSON”，浏览器会下载你的预设备份文件。
  2. 导入 JSON：点击“导入 JSON”，选择之前导出的文件，预设会回到“我的预设”里。

Prompt QA

Prompt 质量检查怎么看

页面底部的“Prompt 质量检查”会提醒关键项是否完整，包括主体、场景、风格、比例、负面规则，以及乱码、水印、Logo 限制。

如果提示“缺失”或“建议补充”，说明这条提示词还有地方没写完整。

Workflow

推荐工作流程

  1. 先选一个最接近目标的预设。
  2. 修改广告文案，比如主标题、副标题、优惠信息。
  3. 修改主体和场景，让画面贴近你的业务。
  4. 根据需要补充风格、光线和画质。
  5. 看“Prompt 质量检查”是否基本通过。
  6. 选择“结构化中文”，点击“复制完整 Prompt”。
  7. 粘贴到 AI 绘图或视频工具里测试。
  8. 如果效果不错，回来点击“保存为我的预设”，最后“导出 JSON”备份。

FAQ

常见问题

复制后去哪里粘贴？

粘贴到你使用的 AI 绘图、AI 视频或海报生成工具里，比如支持中文提示词的图像生成工具、视频生成工具或工作流节点里。

为什么要写负面规则？

因为 AI 生成广告图时容易出现乱码、错别字、水印、奇怪 Logo、二维码等问题。负面规则就是提前告诉 AI：这些东西不要出现。

我的预设会同步到手机或另一台电脑吗？

不会。当前网站没有账号系统，也没有服务器同步。换电脑前请先“导出 JSON”，再到新电脑里“导入 JSON”。

这个工具是不是滴滴官方的？

不是。这是个人网页工具模板，不是滴滴官方产品，也不包含任何官方授权关系。
