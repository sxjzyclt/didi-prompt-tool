# Didi Prompt Tool 新手操作指南

这是一份给第一次使用者看的操作说明。你不需要懂代码，也不需要安装软件，打开网页就能用。

在线地址：

```text
https://sxjzyclt.github.io/didi-prompt-tool/
```

## 这个网站是干什么的

Didi Prompt Tool 是一个广告创意 Prompt 工作台。

Prompt（提示词）就是你发给 AI 绘图或 AI 视频工具的文字说明。这个网站可以帮你把广告画面里的主体、场景、构图、风格、文案和负面规则整理成一段完整提示词。

适合用来做：

- 小程序 Banner
- 活动页主视觉
- 电商促销海报
- 本地生活门店广告
- 游戏宣传图
- 旅游宣传图
- 短视频开屏图
- 文生视频首帧和镜头规则

## 第一次使用

1. 打开网页：`https://sxjzyclt.github.io/didi-prompt-tool/`
2. 点击页面上的“进入工作台”。
3. 左侧选择一个预设，比如“小程序广告”或“电商促销”。
4. 中间编辑每个模块里的内容。
5. 右下方查看“完整提示词”。
6. 点击“复制完整 Prompt”，粘贴到你常用的 AI 绘图或 AI 视频工具里。

## 选择预设

左侧是“预设中心”。

你可以直接点击预设卡片，也可以用搜索框搜索关键词，比如：

- `旅游`
- `电商`
- `游戏`
- `科技`
- `本地生活`

上方分类按钮可以快速筛选不同广告类型。

如果你自己保存过预设，可以点击“我的预设”查看。

## 文生图和文生视频怎么选

页面顶部有两个模式：

- 文生图（Text-to-Image）：适合生成海报、Banner、主视觉、封面图。
- 文生视频（Text-to-Video）：适合生成短视频、开屏动态素材、镜头运动提示词。

如果你只是想做一张广告图，选“文生图”。

如果你想让 AI 生成一段视频，选“文生视频”。选择后会多出视频长度、镜头运动、动态强度等设置。

## 怎么填写内容

每个模块可以按下面思路填。

基础设定：

- 画面类型 / 预设名称：给这条提示词起个名字。
- 画面比例：常见有 `16:9`、`9:16`、`1:1`、`4:5`。

画面内容：

- 主体：画面里最重要的人、物、产品或角色。
- 场景：背景环境，比如城市街区、商场门口、科技展厅。
- 动作 / 姿态：人物或物体正在做什么。
- 服装：人物穿什么。
- 道具：优惠券、手机、礼盒、按钮、金币等。
- 构图：标题放哪，主体放哪，画面横版还是竖版。
- 镜头：近景、中景、广角、低角度、电影感镜头等。

视觉风格：

- 视觉风格：商业插画、3D 卡通、轻写实广告、电影海报等。
- 光线 / 色彩：暖橙色、科技蓝、蓝紫霓虹、高级灰等。
- 光线细节：柔和日光、影棚布光、边缘轮廓光等。
- 画质细节：高清、干净背景、边缘清晰、无噪点等。
- 字体 / 标语规则：告诉 AI 标题字要清楚、有设计感、不要乱码。

广告文案：

- 主标题：广告最醒目的大字。
- 副标题：补充说明。
- 价格 / 优惠信息：比如“最高抵扣15元”。
- 按钮文案：比如“立即领取”“马上抢”。
- 完整标语：把主标题、副标题和按钮文案组合成一句完整广告语。

负面规则：

这里是告诉 AI 不要生成什么。

建议保留这些限制：

- 不要乱码
- 不要错别字
- 不要水印
- 不要二维码
- 不要真实品牌 Logo
- 不要个人信息

## 提示词库怎么用

在“画面内容”和“视觉风格”模块下面，有很多小标签。

点击标签可以把它添加到当前字段里。

比如你在“主体”里点击“年轻情侣”，它会自动加入主体描述。

如果点错了，可以点标签里的减号移除。

## 输出格式怎么选

完整提示词区域上方有 3 种格式：

- 结构化中文：最适合新手，分段清楚。
- 一行中文：适合复制到只支持单行输入的工具。
- SD 分段：适合 Stable Diffusion 这类区分 Positive Prompt 和 Negative Prompt 的工具。

不知道选哪个时，默认用“结构化中文”。

## 复制按钮怎么用

下方有几个复制按钮：

- 复制完整 Prompt：复制当前完整提示词，最常用。
- 复制 Positive：只复制正向提示词。
- 复制 Negative：只复制负面提示词。

新手建议优先用“复制完整 Prompt”。

## 保存自己的预设

如果你改出了一套常用模板，可以点击“保存为我的预设”。

保存后，它会出现在左侧“我的预设”分类里。

注意：保存的数据在浏览器本地，不是在服务器上。

localStorage（浏览器本地存储）被清理后，“我的预设”可能会消失。

## 导出和导入 JSON

JSON（数据备份文件）可以理解成“我的预设备份”。

导出 JSON：

1. 点击“导出 JSON”。
2. 浏览器会下载一个文件。
3. 这个文件就是你的预设备份。

导入 JSON：

1. 点击“导入 JSON”。
2. 选择之前导出的 JSON 文件。
3. 导入成功后，预设会出现在“我的预设”里。

建议你每次整理出重要预设后，都导出一次 JSON，避免换电脑或清理浏览器后丢失。

## Prompt 质量检查怎么看

页面底部有“Prompt 质量检查”。

它会检查这些内容是否完整：

- 是否填写主体
- 是否填写场景
- 是否填写风格
- 是否填写比例
- 是否有负面规则
- 是否包含乱码、水印、Logo 限制

如果提示“缺失”或“建议补充”，说明这条提示词还有地方没写完整。

## 推荐工作流程

新手可以按这个顺序来：

1. 先选一个最接近目标的预设。
2. 修改广告文案，比如主标题、副标题、优惠信息。
3. 修改主体和场景，让画面贴近你的业务。
4. 根据需要补充风格、光线和画质。
5. 看“Prompt 质量检查”是否基本通过。
6. 选择“结构化中文”。
7. 点击“复制完整 Prompt”。
8. 粘贴到 AI 绘图或视频工具里测试。
9. 如果效果不错，回来点击“保存为我的预设”。
10. 最后点击“导出 JSON”备份。

## 常见问题

### 1. 复制后去哪里粘贴？

粘贴到你使用的 AI 绘图、AI 视频或海报生成工具里。

比如你可以粘贴到支持中文提示词的图像生成工具、视频生成工具或工作流节点里。

### 2. 为什么要写负面规则？

因为 AI 生成广告图时容易出现乱码、错别字、水印、奇怪 Logo、二维码等问题。

负面规则就是提前告诉 AI：这些东西不要出现。

### 3. 我的预设会同步到手机或另一台电脑吗？

不会。

当前网站没有账号系统，也没有服务器同步。你的预设只存在当前浏览器里。

如果要换电脑，请先“导出 JSON”，再到新电脑里“导入 JSON”。

### 4. 清空当前编辑会删除我的预设吗？

不会。

“清空当前编辑”只会清空当前正在填写的内容，不会删除已经保存的“我的预设”。

### 5. 导出 JSON 是不是上传到网上？

不是。

导出 JSON 是把备份文件下载到你的电脑，不会上传服务器。

### 6. 这个工具是不是滴滴官方的？

不是。

这是个人网页工具模板，不是滴滴官方产品，也不包含任何官方授权关系。

## 一句话记住

先选预设，再改内容，检查质量，复制 Prompt，最后保存并导出备份。
